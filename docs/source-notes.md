# Источники данных

Основная информация для сайта взята с официального сайта школы:

- https://shkola-darit-nadezhdu.gosuslugi.ru/
- https://shkola-darit-nadezhdu.gosuslugi.ru/glavnoe/o-tsentre/
- https://shkola-darit-nadezhdu.gosuslugi.ru/glavnoe/kontakty/
- https://shkola-darit-nadezhdu.gosuslugi.ru/roditelyam-i-uchenikam/raspisanie/

- https://shkola-darit-nadezhdu.gosuslugi.ru/nasha-shkola/administratsiya/

В проект перенесены:

- название образовательной организации;
- адрес, телефон, электронная почта и режим работы;
- сведения о директоре и администрации;
- направления обучения;
- изображения здания, директора, карты и расписания;
- новости и информационные блоки по мотивам официального сайта.

Часть расписания в таблицах добавлена как демонстрационные записи для работы администратора. Официальные изображения расписания сохранены в разделе "Расписание" и в папке `public/assets`.
